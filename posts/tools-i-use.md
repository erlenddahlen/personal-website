# Tools I Use Every Day

As a developer, I've settled into a set of tools that make my workflow smooth. Here's what I reach for daily.

## Editor

VS Code is my go-to. It's fast enough, the extension ecosystem is massive, and it handles everything from Python to Terraform without breaking a sweat. A few extensions I can't live without:

- **GitLens** — inline blame and history
- **Prettier** — consistent formatting without thinking
- **GitHub Copilot** — surprisingly useful for boilerplate

## Terminal

I use iTerm2 with zsh and a minimal prompt. No fancy themes — just the current directory and git branch. The less visual noise, the better.

## Git workflow

I keep it simple: short-lived feature branches, small commits, and squash merges. `git log --oneline` should read like a changelog.

## Notes

Plain markdown files in a folder. No apps, no sync issues, no vendor lock-in. I grep my notes more than I browse them.

## The common thread

Every tool I stick with shares the same quality: it stays out of my way. The best tools are the ones you stop noticing.
